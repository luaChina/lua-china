# Lua 中国开源社区

[![Build Status](https://travis-ci.org/luaChina/lua-china.svg?branch=master)](https://travis-ci.org/luaChina/lua-china)

## 技术栈

### 前端

* 服务端渲染 Nuxt.js

* bootstrap4

* fontawesome

### 后端

* Kong API 网关

* openresty Nana 框架

* mysql5.7

* redis

## 持续集成

* travis-ci

## 部署

`docker run -p 80:80 --name=lua-china -d horan/lua-china:0.1`
